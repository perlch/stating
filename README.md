# Strategy Territory Game

A lightweight, vanilla web-based strategy game inspired by State.io. Command your units, capture territories, and dominate the map in this minimalist tactical experience.

## Features

- **Pure Vanilla Tech**: Built entirely with HTML5, CSS3, and Vanilla JavaScript. No frameworks or external libraries required.
- **Portable & Lightweight**: The entire game can be run from a single file. Just download and open it in any modern browser to start playing.
- **Deep Customization**: Extensive game settings including Fog of War, match timers, unit growth rates, and visual themes.

## Future Roadmap

- **Multiplayer Mode**: Real-time battles against other players.
- **Mobile Support**: Native .apk version for Android using Apache Cordova.
- **Advanced AI**: Improved computer difficulty levels and behaviors.

## Installation

1. Clone or download the repository.
2. Open `index.html` in your web browser.
3. No build process or local server is necessary.
